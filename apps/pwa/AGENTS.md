# FitTracker PWA

Next.js PWA for the fitness hub: workouts, bioimpedance, water, AI planning, dashboard.

## Stack

- Next.js 14 (App Router), Tailwind CSS, React 18

## Build / Run

```bash
npm run dev
# or from root: npm run pwa
```
