# FitTracker Mobile

React Native (Expo) app for the fitness hub: workouts, bioimpedance, water, AI planning.

## Stack

- Expo SDK 52, React Navigation, React Native

## Build / Run

```bash
npx expo start
# or from root: npm run mobile
```
